The Dock Forum

The Dock Forum is a lightweight, anonymous internal forum built to surface **safety, operations, and general concerns** in a low-friction, psychologically safe way.

It is intentionally designed as a **frontend MVP** with shared, real-time data and locked backend security.

Features

* Anonymous thread posting (no accounts, no identities)
* Category-based posts (Safety, Operations, General)
* Shared posts across all users (Firebase Firestore)
* Newest-first thread feed
* Persistent storage
* Locked Firestore rules (no edits or deletes from clients)
* Static deployment (GitHub Pages)

Tech Stack

* HTML, CSS, JavaScript
* Firebase Firestore (shared datastore)
* GitHub Pages (static hosting)

Purpose

This project demonstrates how to design and deploy a secure, anonymous feedback system with minimal infrastructure, focusing on usability, trust, and scalability.
